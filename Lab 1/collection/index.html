üíª Technical Implementation

The server is built using Python's socket module for low-level network communication. It parses HTTP requests, determines the requested resource, and sends appropriate HTTP responses with correct headers and content types.

Supported Operations:

  • File Serving: HTML, PNG, PDF with correct MIME types
  • Directory Listing: Auto-generated HTML pages for directories
  • Error Handling: 404 Not Found, 403 Forbidden, 405 Method Not Allowed
  • Security: Path traversal protection
